# Monoglot
### by Team Aggressive Aluminum

**Monoglot** is a documentation tool for constructed languages.

---

Build with:

```
gradle assemble
```

The jar will be at `build/lib/Monoglot.jar`.

Build and run with:

```
gradle runShadow
```

---

Build for mac with:

```
gradle createDmg
```

Then run what that spits out.
